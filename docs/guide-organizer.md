# イベント主催者ガイド

カードゲームイベントのマッチング管理を行うための操作ガイドです。

---

## はじめかた

1. **アプリを開く** — https://soichirow.github.io/matching-kanri/
2. **テーブルを追加** — ヘッダーの「＋テーブル」ボタン → ダイアログで卓数と人数を選択
3. **プレイヤーを追加** — 「＋プレイヤー」ボタン → ダイアログで人数を選択、またはExcelから一括入力
4. **マッチング** — 画面下部の「マッチング」ボタンを押すだけ

---

## イベントの流れ

### 準備

- **テーブル追加**: 「＋テーブル」ボタン → ダイアログで卓数(1〜10)と対戦人数(2〜8人)を選択して追加。デフォルトは4人卓
- **プレイヤー追加**: 「＋プレイヤー」ボタン → ダイアログで人数を選択して一括追加。1人の場合は名前・グループを入力するダイアログが表示。「追加して続ける」で連続追加も可能
- **Excel一括入力**: クリップボードアイコンをクリック。Excelの列をそのまま貼り付け（A列:名前、B列:グループ、C列:ブラケット）

### マッチング

1. 「マッチング」ボタンを押す → テーブルに仮配置が表示（黄色枠）
2. 内容を確認して「全確定」→ 対戦開始
3. 気に入らない場合は再生成ボタン（ループアイコン）でやり直し
4. 特定のテーブルだけ取り消すこともできます

### ラウンド中

- テーブルカードに対戦中のプレイヤーが緑枠で表示
- 点数記録が有効なら、テーブルカード内に入力欄が表示
- 対戦終了はテーブルカードの「対戦終了」ボタン、または画面下部の「全終了」

### 次のラウンド

対戦終了後、再び「マッチング」を押すと過去の対戦を考慮した新しい組み合わせが生成されます。

---

## 参加者への共有

1. 右上の設定ボタンをクリック
2. 「イベント共有」セクションの「イベントを共有する」をクリック
3. **参加コード**（4桁の英数字）が発行されます
4. 参加者への伝え方:
   - 口頭で「コード○○○○を入力してね」
   - QRコードボタンを押して画面を見せる
   - URLをLINEグループ等で送る

共有後は管理画面での操作が自動的に参加者のスマホに反映されます。

---

## グループとブラケット

### グループ
同じグループのプレイヤーは **同じテーブルになりにくく** なります。
- 用途: チーム分け、友人同士を別卓に、レベル帯で分散
- A〜Eのプリセット + 任意のグループ名を設定可能

### ブラケット
同じブラケットのプレイヤー **だけで** マッチングします。
- 用途: 予選リーグ、レベル別リーグ
- 設定 → 「ブラケット機能」をONにすると有効

> グループ = 「一緒にしない」、ブラケット = 「この中だけ」

---

## 点数の記録

1. 設定 → 「点数を記録する」をON
2. 対戦中のテーブルカードに±の入力欄が表示
3. 対戦終了時に自動で累計に加算
4. ヘッダーの順位表ボタンで全体順位を確認
5. コピー・CSVダウンロードで結果を出力

---

## 設定項目の説明

| 設定 | 内容 |
|------|------|
| イベント名 | ヘッダーと参加者ビューに表示される名前 |
| テーマ | ダーク/ライトの切替 |
| ラウンド数を表示 | ステータスバーに現在のラウンド番号を表示 |
| ブラケット機能 | 同ブラケット内でのみマッチング |
| 少人数卓の連続を避ける | 前回ショート卓だった人を今回のショート卓に入れにくくする |
| 点数を記録する | VP入力欄と順位表を有効化 |
| 点数をマッチングに反映 | 累計点数が近い人を同卓にしやすくする |
| 最低人数 | テーブルサイズ未満でも配席する最低人数（0=テーブルサイズと同じ） |

---

## テーブル操作

- **ドラッグ**: PCではテーブルカードをドラッグして自由配置（グリッドにスナップ）
- **設定ボタン**（歯車アイコン）: テーブルの人数変更、名前変更、メンバー手動編集、削除
- **解散**: 対戦を取り消してプレイヤーを待機に戻す（対戦履歴も消去）
- **配置物**: 設定 → 「配置物」で壁やモニターなどのオブジェクトを配置可能

---

## ショートカット

| 操作 | 動作 |
|------|------|
| Ctrl+Z | 元に戻す |
| Shift+クリック | プレイヤーを範囲選択 |
| プレイヤー名をタップ | 編集ダイアログを開く |

---

## 新しいイベントを始める

設定 → 「新しいイベントを始める」で、テーブル配置を維持したままプレイヤーと対戦履歴をリセットできます。毎週同じ会場で開催する場合に便利です。

---

## データについて

- データはブラウザの **localStorage** に保存されます
- 同じブラウザ・同じURLでアクセスすれば前回のデータが残っています
- **別のブラウザやシークレットモードではデータは見えません**
- 重要なデータは順位表の「CSVダウンロード」でバックアップを推奨します
